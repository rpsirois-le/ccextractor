Quick and dirty walk a directory and parse text files for common credit card numbers and dump them out
console output gives a bit more specific information

Sources cited inline where used

This is a Python 3 script.

Usage: `python3 ccextractor.py path/to/files`

Optionally pass -o with an output filename to dump credit card numbers to:
`python3 ccextractor.py path/to.files -o ./output.txt`






Test files included for funsies.
